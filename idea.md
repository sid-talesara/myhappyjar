# My Happy Jar — Idea

## Overview
A mobile-first PWA where users write daily gratitude/happy notes on virtual sticky notes, fold them, and drop them into a glass jar. Notes accumulate over a year into a collectible jar.

---

## Core Features

**Sticky Note Entry**
- Daily note: max 150 chars
- Randomized note color per day, or user-selectable (pastel palette)
- Optional: attach 1 image or 1 audio note per entry
- One note per day (can edit until midnight)

**The Jar**
- Translucent glass jar UI — folded notes visible inside
- Notes animate: fold → drop into jar
- Date stamp on each folded note (visible on hover/tap)
- Jar fills up visually as year progresses

**Jar Collection**
- One jar per calendar year (auto-created Jan 1)
- User can name/rename their jar
- Past jars accessible as a shelf/archive view
- Tap a folded note inside jar → unfolds and shows content

**Note Detail View**
- Full note text
- Date
- Color
- Attached image/audio (if any)

---

## Platforms
- Mobile PWA (primary)
- Web (responsive, same codebase)

---

## Out of Scope (v1)
- Social sharing
- Reminders/notifications (v2)
- Multiple notes per day
- Collaboration

---

## Success Metric
User opens app and adds a note for 7 consecutive days.
